# 제목
노인-방언 환경에서 Test-Time Personalization 기반 한국어 음성인식 성능 개선

## 초록
- 문제: 범용 STT는 노인/방언 환경에서 오류율이 높음
- 방법: 상위 레이어 경량 TTT + 지속 적응(P2)
- 결과: B0/B1 대비 P1/P2 개선, 통계적 유의성 검증
- 의의: 제한 GPU 환경에서 재현 가능한 개인화 STT 프로토콜 제시

## 1. 서론
- 고령층/방언 환경의 실제 도입 문제
- 기존 범용 STT의 한계
- 본 논문의 기여점 3가지

## 2. 방법
### 2.1 데이터
- AI Hub 노인/방언 데이터
- 화자 독립 분할과 층화 샘플링 규칙

### 2.2 모델
- B0/B1/P1/P2 정의
- 상위 레이어 부분 업데이트 전략

### 2.3 평가
- WER/CER
- 통계검증(Wilcoxon, Cliff's delta)
- UI 실험 지표(완료시간/수정횟수)

## 3. 실험 설정
- 하드웨어, 학습 조건
- 캘리브레이션 문장 수/적응 스텝
- 아블레이션 축(top-k/step/calibration)

## 4. 결과
### 4.1 메인 결과
- B0→B1→P1→P2 비교표
- 평균/분포 그래프

### 4.2 하위집단 분석
- 지역/발화유형/연령대 별 개선

### 4.3 아블레이션
- 최적 설정 및 트레이드오프

### 4.4 UI 검증
- 고령층 친화 UI 가설 검증 결과

## 5. 논의
- 실패 사례
- 한계(데이터 범위/노이즈/일반화)
- 현장 도입 시 고려사항

## 6. 결론
- 핵심 성과 요약
- 후속 연구(지역 확장, 번역 결합)
